## Getting Started

1.Clone the repo
   ```sh
   git clone https://github.com/yantsev1ch/todo-app.git
   ```
2.Install packages
   ```sh
   yarn
   ```
3.Start project
   ```sh
   yarn start
   ```

### Stack:
1. React - https://reactjs.org/
2. MobX - https://mobx.js.org/
3. Typescript - https://www.typescriptlang.org/
